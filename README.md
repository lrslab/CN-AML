# CN-AML
nanopore whole-genome sequencing of cytogenetically normal acute myeloid leukaemia 
1. Copy number and single nucleotide variation detection
2. Methylation calling
3. Epitype clustering
4. Whole-genomw 5mC profiling (100b bin, 100k bin, chromosomes)
5. Regional 5mC profiling (promoters, CpG island and flanking regions)
6. Repetitive elements 5mC and copy number profiling (rDNA and others)
